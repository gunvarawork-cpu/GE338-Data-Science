## พื้นที่ศึกษา

เลือกพื้นที่เป็นบริเวณอำเภอสอง จังหวัดแพร่ เนื่องจากอย่างแรกที่เลือกคือภาคเหนือเพราะภาคเหนือมีภูมิประเทศที่เป็นป่าและภูเขาส่วนใหญ่ ทำให้หากจะวางแผนการจัดการที่ดินในฤดูแล้ง ภาคเหนือก็เป็นตัวเลือกที่เหมาะสม สามารถดูพวกพืชพรรณและสังเกตได้ง่ายหากพืชพรรณบริเวณไหนหายไป โดยเจาะจงเป็นอำเภอสอง เพราะเป็นพื้นที่ที่มักได้รับผลกระทบหนักจากการขาดแคลนน้ำอุปโภคบริโภคและน้ำเพื่อการเกษตร มีปัญหาภัยแล้งบ่อย

คำถามวิจัย
- พื้นที่บริเวณไหนของอำเภอสองที่แห้งแล้งที่สุด เนื่องจากมีป่าเกือบทั้งอำเภอ

## เลือกดาวเทียมและช่วงเวลา

ใช้เป็นดาวเทียม Sentinel-2 เนื่องจาก sentinel-2 มีความละเอียดที่สูง เหมาะต่อการดูพื้นที่ในระดับอำเภอ
